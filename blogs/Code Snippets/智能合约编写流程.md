# 智能合约编写流程

```bash
// 创建项目目录，安装基本包
mkdir epic-nfts
cd epic-nfts
npm init -y
npm install --save-dev hardhat
```

```bash

// 执行hardhat，初始化环境，默认都yes
npx hardhat
```

```bash
// 安装zeppelin 可升级合约包
npm install --save-dev @openzeppelin/hardhat-upgrades
```

```bash
// 运行默认环境，测试环境正确
npx hardhat run scripts/sample-script.js
```
