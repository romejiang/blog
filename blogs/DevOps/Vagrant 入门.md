# Vagrant 入门

**下载安装**

vagrant 依赖 virtualbox，需要先安装virtualbox。然后到在下面地址下载相应的安装包。

[https://www.vagrantup.com/downloads.html](https://www.vagrantup.com/downloads.html)

**创建vagrant项目**

vagrant 类似 npm 项目目录下都有一个项目的配置文件 Vagrantfile。初始化项目目录的命令

vagrant init
